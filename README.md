# Финансовый Трекер

Приложение для отслеживания личных финансов с аутентификацией, графиками расходов и навигацией.

## Архитектура проекта

### Структура папок

```
src/
├── components/     # Переиспользуемые компоненты
│   ├── Auth.jsx        # Форма регистрации/авторизации
│   ├── Dashboard.jsx   # Главная страница с графиками и карточками
│   └── Menu.jsx        # Нижняя навигационная панель
├── context/        # Контексты для управления состоянием
│   └── AuthContext.jsx # Контекст аутентификации
├── pages/          # Страницы приложения
│   ├── LoginPage.jsx   # Страница входа
│   └── DashboardPage.jsx # Страница дашборда
└── App.jsx         # Главный компонент с роутингом
```

### Основные компоненты

- **Auth.jsx**: Компонент для входа и регистрации пользователей
- **Dashboard.jsx**: Отображает графики расходов и карточки с категориями (например, "Одежда -1500 руб.")
- **Menu.jsx**: Нижняя менюшка с кнопками "Профиль" и "Выход" (только UI)

### Страницы

- **LoginPage.jsx**: Страница авторизации, перенаправляет на дашборд после входа
- **DashboardPage.jsx**: Основная страница с данными и навигацией

### Управление состоянием

- **AuthContext.jsx**: Контекст для хранения состояния пользователя и функций аутентификации

### Роутинг

В App.jsx настроен роутинг с использованием react-router-dom:

- `/login` - страница авторизации
- `/dashboard` - главная страница

## Запуск проекта

```bash
npm install
npm run dev
```
